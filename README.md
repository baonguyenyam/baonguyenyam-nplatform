# Setup & Running

### Create .env.local 

Copy `.env.sample` to `.env`

Run this command:

```bash
cp .env.sample .env
```

### Add Next Auth Key 

Run this command 

```bash
npx auth secret
```
Then => Yes

### Install component

Run this command

```bash
yarn
```

### Run the app

Run this command 

```bash
sh app.sh
```
